# InterviewAI 🧠

AI-powered mock interview coach. Practice, get instant feedback, and master your interview skills.

## Features
- 7 industry tracks (Software Engineering, Product Management, Marketing, Finance, Design, Data Science, Sales)
- AI-generated feedback on every answer
- Smart mastery system — correctly answered questions never repeat
- MCQ + Open-ended questions
- Full interview history saved per user

## Tech Stack
- React 18 + Vite
- Anthropic Claude API for AI feedback
- Tailwind-free, pure inline styles

---

## Deploy to Vercel (Recommended — Free)

1. Upload this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → Sign up with GitHub
3. Click **Add New Project** → Import your repo
4. Framework will be auto-detected as **Vite**
5. Click **Deploy** — live in 60 seconds ✅

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173
